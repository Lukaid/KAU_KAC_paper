### 연구방법

- 우선 네이버 뉴스를 통해 크롤링 진행
- 네이버 뉴스 포맷을 제공하는 기사만 크롤링
- 네이버 뉴스는 하나의 기간 검색에 최대 4000건의 기사만을 제공
- 따라서 기간을 잘게 나눌수록 유리함
- 일주일 간격으로 기간을 나누었음 (아래 예시)
- 
    ```python
    day_start = ["2019.12.01", "2019.12.08", "2019.12.15", .....
    day_end = ["2019.12.07", "2019.12.14", "2019.12.21", .....
    ]
    ```
